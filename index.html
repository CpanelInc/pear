cPanel PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover cpanelinc.github.com/pear
Listing available packages: 
pear remote-list -c Cpanel
Installing a package: 
pear install Cpanel/package_name
Installing a specific version/stability: 
pear install Cpanel/package_name-1.0.0
pear install Cpanel/package_name-beta
Receiving updates via a feed: 
http://cpanelinc.github.com/pear/feed.xml
